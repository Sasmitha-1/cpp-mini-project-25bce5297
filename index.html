Project Overview

A menu-driven mini banking system built using C++, object-oriented programming, STL, file handling with fstream, and deployed online using WebAssembly.

Features

  • Create Account Unique account number with initial balance
  • Deposit / Withdraw No overdraft allowed
  • Transaction History Last 5 transactions
  • Reports Total funds and low balance accounts

Classes Used

  • Account Stores account details and balance operations
  • Transaction Stores deposit and withdrawal history
  • BankSystem Handles menu, vectors, reports, and files

Premium banking console powered by C++ and WebAssembly.

This interface presents the DA2 project like a polished banking dashboard while preserving the original C++ application workflow. The program logic runs from main.cpp.
